Project Name: Travel & Tourism Website

This project showcases a responsive travel website designed with a strong focus on User Interface Design (UID) principles. Built using semantic HTML5, modern CSS3 for styling and animations, and JavaScript for dynamic interactivity, it delivers an immersive browsing experience. The layout features intuitive navigation, captivating imagery, and smooth user interactions to simulate a seamless journey planning platform. This repository demonstrates proficiency in core frontend technologies and the ability to translate design concepts into functional code.


Tech Stack : 
HTML ,
CSS  ,
JavaScript

Features :
 Attractive homepage with travel destinations  ,
 Destination details section  ,
 Travel packages display  ,
 Responsive design  ,
 Smooth navigation between pages
